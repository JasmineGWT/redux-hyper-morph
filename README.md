# redux-yo


## how


### install
Link your medux module into this repo. Change to your medux repo and run:
```
npm link
```
Switch back to this repo and run
```
npm link medux
```
Then:
```
npm install
```

## develop

```
npm start
```

browse to <http://localhost:9966/>.

## test

```
npm test
```

## deploy

```
npm run deploy
```
